# 调证材料文档格式规范

## 文档概述
调证材料是律师向相关部门申请调取证据的法律文书，主要用于调查取证场景。

## 模板位置
`../legal_templates/2 9 调证材料.docx`

## 快速生成命令
```bash
# 生成正式文书
python ../scripts/create_investigation_doc.py

# 生成空白模板
python ../scripts/create_blank_investigation_doc.py
# 或使用统一生成器
python ../scripts/create_all_blank_templates.py civil
```

## 空白模板生成规则
生成空白模板时，占位符替换为8个空格（`        `），以保证打印文档宽度足够。

## 格式规范

### 1. 字体规范
| 元素 | 字体 | 字号 |
|------|------|------|
| 标题（"授权委托书"、"律师事务所证明"） | 方正小标宋简体 | 二号（279400 EMU） |
| 正文 | 仿宋 | 三号（177800 EMU） |
| 标签加粗（"委托人："、"受托人："） | 仿宋 | 三号（177800 EMU），粗体 |

### 2. 段落结构

#### 授权委托书部分
- 标题"授权委托书"居中，方正小标宋简体二号字
- "委托人："单独成行，加粗
- "受托人："单独成行，加粗
- 方框选项"□ 自然人："、"□ 法人："
- "委托事项:"单独成行，加粗

#### 律师事务所证明部分
- 标题"律师事务所证明"居中，方正小标宋简体二号字
- 开头："兹证明，因我所 [律师姓名] 律师（执业证号：[证号]，联系电话：[电话]）承办 [委托人] 委托的 [事项] 一案（事），需向贵单位申请调取如下证据、信息材料："
- 调取内容以数字编号列出

#### 签章部分
- "委托人(签名/盖章):" 左对齐
- "年    月    日" 右对齐
- "律师签名："、"律师事务所负责人签名："、"律师事务所（章）"等

### 3. 占位符完整列表

| 占位符 | 含义 | 示例 |
|--------|------|------|
| weitr | 委托人姓名 | 张三 |
| wtrsfz | 委托人身份证号码 | 430124198203045123 |
| dfdsr | 对方当事人 | 李四 |
| anyou | 案由/事项 | 户籍信息查询 |
| dcdw | 调取单位 | 宁乡市公安局 |
| bdqxx | 被调取信息内容 | 李四的户籍信息：430282195607084321 |

### 4. 生成脚本逻辑

```python
# 委托人信息替换
if 'weitr' in full_text:
    for run in para.runs:
        if 'weitr' in run.text:
            run.text = run.text.replace('weitr', '委托人姓名')
        if 'wtrsfz' in run.text:
            run.text = run.text.replace('wtrsfz', '身份证号')
        if 'dfdsr' in run.text:
            run.text = run.text.replace('dfdsr', '对方当事人')
        if 'anyou' in run.text:
            run.text = run.text.replace('anyou', '事项')

# 调取单位替换（dcdw已整合为完整占位符）
if 'dcdw' in full_text:
    for run in para.runs:
        if 'dcdw' in run.text:
            run.text = run.text.replace('dcdw', '调取单位')

# 调取内容替换
if 'bdqxx' in full_text:
    for run in para.runs:
        if 'bdqxx' in run.text:
            run.text = run.text.replace('bdqxx', '被调取的具体信息内容')
```

### 5. 典型案例数据

**案例1：委托律师调取户籍信息**
- 委托人：张三，身份证430124198203045123
- 受托人：陈伟律师，湖南金厚（宁乡）律师事务所
- 调取单位：宁乡市公安局
- 对方当事人：李四
- 被调取信息：李四的户籍信息：430282195607084321

## 注意事项
1. 标题必须使用方正小标宋简体
2. 正文使用仿宋三号
3. 保持原有段落结构
4. 占位符可能分散在多个runs中，需逐个替换
5. 所有占位符（weitr, wtrsfz, dfdsr, anyou, dcdw, bdqxx）均为完整占位符
