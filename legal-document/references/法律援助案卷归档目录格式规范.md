- 民事：`../legal_templates/9 湖南省法律援助案卷归档目录（民事）.docx`
- 刑事：`../legal_templates/8 湖南省法律援助案卷归档目录（刑事）.docx`

```bash

python ../scripts/create_civil_legal_aid_directory.py

# 刑事法律援助目录
python ../scripts/create_criminal_legal_aid_directory.py

# 生成空白模板
python ../scripts/create_blank_civil_legal_aid_directory.py

# 或使用统一生成器
python ../scripts/create_all_blank_templates.py civil    # 民事
python ../scripts/create_all_blank_templates.py criminal # 刑事
```

### 5. 生成脚本逻辑

```python
from docx import Document

def create_legal_aid_directory(template_path, output_path, city_name="宁乡市"):
    """生成法律援助案卷归档目录"""

    shutil.copy(template_path, output_path)
    doc = Document(output_path)

    for para in doc.paragraphs:
        if city_name in para.text and "法律援助案卷归档目录" in para.text:
            for run in para.runs:
                if city_name in run.text:
                    run.text = city_name
                elif "法律援助案卷归档目录" in run.text:
                    run.text = "法律援助案卷归档目录"

    doc.save(output_path)
```

- 质量评估：按目录核查办案规范性
