## 总则

### 基本原则

1. **准确性原则**
   - 事实表述准确无误
   - 法律引用精确到位
   - 数字、日期、人名等信息核实无误

2. **规范性原则**
   - 符合法律法规要求
   - 符合官方文书样式
   - 使用规范的法律术语

3. **完整性原则**
   - 要素齐全，无遗漏
   - 逻辑完整，条理清晰
   - 结论明确，表述完整

4. **简洁性原则**
   - 语言精练，避免冗余
   - 重点突出，层次分明
   - 一文一事，主题明确

## 文书结构规范

### 一、诉讼文书结构

#### 1. 民事起诉状

**标准结构**
```
【标题】民事起诉状（居中、二号字体、加粗）

【当事人信息】
原告：姓名，性别，出生日期，民族，职业，住址，联系方式
被告：姓名，性别，出生日期（或年龄），住址，联系方式

【诉讼请求】
1. 请求事项一
2. 请求事项二
...

【事实与理由】
[案件事实经过]
[起诉理由]
[法律依据]

【证据清单】
证据一：[证据名称]，证明[证明目的]
证据二：[证据名称]，证明[证明目的]
...

【此致】
XX人民法院

【落款】
起诉人：[签名]
[日期]
```

**写作要点**

1. **当事人信息**
   - 原告信息要完整，便于法院联系
   - 被告信息要明确，便于法院送达
   - 法人单位要写明法定代表人

2. **诉讼请求**
   - 具体明确，避免模糊表述
   - 逐项列明，条理清晰
   - 有数字的写明具体金额
   - 可分项列出：本金、利息、违约金等

   **示例对比**
   ```
   ❌ 错误：请求法院依法判决
   ✅ 正确：
   1. 判令被告偿还原告借款本金人民币10万元；
   2. 判令被告支付自2024年1月1日起至实际清偿之日止的利息（按年利率3.45%计算）；
   3. 本案诉讼费用由被告承担。
   ```

3. **事实与理由**
   - 按时间顺序叙述事实
   - 交代清楚人物、时间、地点、经过、结果
   - 引用相关法律条文
   - 语言客观，避免情绪化表达

   **叙述结构**
   ```
   第一段：背景介绍（当事人关系、合同签订等）
   第二段：纠纷发生（违约事实、侵权行为等）
   第三段：损害后果（损失情况、影响等）
   第四段：法律依据（引用法条）
   第五段：起诉理由（为何诉讼）
   ```

4. **证据清单**
   - 列明证据名称
   - 说明证明目的
   - 注明证据来源

#### 2. 答辩状

**标准结构**
```
【标题】民事答辩状

【当事人信息】
答辩人：[基本信息]
被答辩人：[基本信息]

【答辩意见】
针对原告诉讼请求一：同意/不同意/部分同意，理由如下：...
针对原告诉讼请求二：...

【事实与理由】
[答辩事实]
[反驳理由]
[法律依据]

【证据清单】
...

【此致】
XX人民法院

【落款】
答辩人：[签名]
[日期]
```

**写作要点**

1. **答辩意见**
   - 逐项回应原告的诉讼请求
   - 明确表态：同意、不同意或部分同意
   - 简要说明理由

2. **事实与理由**
   - 可承认原告陈述的真实事实
   - 重点反驳不实之处
   - 提出新的抗辩事实
   - 引用支持答辩的法律条文

### 二、合同协议结构

#### 1. 标准合同结构

```
【标题】XX合同（居中、加粗）

【合同编号】（可选）

【当事人信息】
甲方：[名称/姓名]
住所/住址：[地址]
法定代表人/负责人：[姓名]
联系方式：[电话]

乙方：[名称/姓名]
...

【前言】（可选）
鉴于...，双方经协商一致，达成如下协议：

【正文条款】
第一条 [条款标题]
[条款内容]

第二条 [条款标题]
[条款内容]

...

第N条 [条款标题]
[条款内容]

【签字盖章】
甲方（盖章）：          乙方（盖章）：
法定代表人（签字）：    法定代表人（签字）：
日期：                  日期：
```

**必备条款**

1. **当事人条款**
   - 明确双方主体身份
   - 写明联系方式
   - 法人单位要有法定代表人

2. **标的条款**
   - 明确合同标的物
   - 详细描述标的特征
   - 明确数量、质量标准

3. **价款或报酬条款**
   - 明确金额（大写和小写）
   - 明确支付方式
   - 明确支付时间

4. **履行条款**
   - 明确履行期限
   - 明确履行地点
   - 明确履行方式

5. **违约责任条款**
   - 明确违约情形
   - 明确违约责任
   - 明确赔偿标准

6. **争议解决条款**
   - 协商、调解、仲裁或诉讼
   - 明确管辖法院或仲裁机构

#### 2. 借款合同

**特殊条款**

1. **借款金额**
   ```
   借款金额为人民币（大写）壹拾万元整（小写：￥100,000元）。
   ```

2. **借款期限**
   ```
   借款期限自2025年1月1日起至2025年12月31日止，共12个月。
   ```

3. **利息约定**
   ```
   借款利率为年利率3.45%，利息自借款发放之日起计算。
   利息支付方式：按季支付，每季末月的20日前支付当季利息。
   ```

4. **还款方式**
   ```
   还款方式：到期一次性还本付息。
   还款账户：开户行XX银行，账号XXXX，户名XX。
   ```

5. **违约责任**
   ```
   借款人未按期归还借款的，应按逾期金额每日万分之五支付违约金。
   借款人未按约定用途使用借款的，贷款人有权提前收回借款。
   ```

### 三、婚姻家庭文书结构

#### 1. 离婚协议书

**标准结构**
```
【标题】离婚协议书

【当事人信息】
男方：姓名，身份证号，住址
女方：姓名，身份证号，住址

【离婚意愿】
男方XX与女方XX于X年X月X日在XX民政局登记结婚。现因...，双方自愿离婚。

【子女抚养】
双方婚生子/女XX（身份证号：...），离婚后由XX抚养。
抚养费：每月X元，由XX支付至XX账户，至子女年满18周岁止。
探望权：...

【财产分割】
1. 房产：位于XX的房屋（房产证号：...）归XX所有，...
2. 车辆：车牌号XX的车辆归XX所有，...
3. 存款：双方名下存款共计X元，XX分得X元，XX分得X元。
4. 其他财产：...

【债务处理】
双方共同债务：...
债务分担：...

【其他约定】
...

【签署】
男方（签名）：    女方（签名）：
日期：            日期：
```

## 语言表达规范

### 一、基本要求

1. **使用书面语**
   ```
   ❌ 口语：我要告他
   ✅ 书面：原告依法向人民法院提起诉讼
   ```

2. **用词准确**
   ```
   ❌ 不准确：大概、可能、差不多
   ✅ 准确：约、可能（需注明推测依据）、约
   ```

3. **表述清楚**
   - 避免歧义
   - 避免模糊
   - 避免多余修饰

### 二、法律术语使用

1. **当事人称谓**

   | 诉讼类型 | 称谓 |
   |---------|------|
   | 民事诉讼 | 原告、被告、第三人 |
   | 刑事诉讼 | 被害人、被告人、犯罪嫌疑人 |
   | 行政诉讼 | 原告、被告 |
   | 合同 | 甲方、乙方 或 出租人、承租人等 |

2. **常用法律术语**

   | 术语 | 含义 | 示例 |
   |-----|------|-----|
   | 当事人 | 参与法律关系的主体 | 合同当事人 |
   | 标的 | 法律行为指向的对象 | 合同标的物 |
   | 违约 | 违反合同约定 | 被告构成违约 |
   | 侵权 | 侵害他人合法权益 | 构成侵权行为 |
   | 诉讼时效 | 向法院请求保护权利的期限 | 诉讼时效期间 |
   | 管辖 | 法院受理案件的范围 | 属于本院管辖 |

3. **数字表述**

   ```
   金额：人民币壹拾万元整（￥100,000元）
   日期：2025年3月22日
   百分比：年利率3.45%（百分之三点四五）
   期限：自2025年1月1日起至2025年12月31日止
   ```

### 三、常用句式

1. **事实陈述句**
   ```
   XX于X年X月X日与XX签订《XX合同》。
   XX未按合同约定履行付款义务。
   XX的行为已构成违约。
   ```

2. **法律引用句**
   ```
   根据《中华人民共和国民法典》第X条规定，...
   依据《中华人民共和国民事诉讼法》第X条之规定，...
   ```

3. **诉讼请求句**
   ```
   请求判令被告偿还借款本金人民币X元。
   请求确认XX合同无效。
   请求驳回原告的诉讼请求。
   ```

## 格式排版规范

### 一、页面设置

- 纸张：A4纸
- 页边距：上下2.54cm，左右3.17cm
- 行距：1.5倍行距或固定值28磅
- 字体：
  - 标题：黑体，小二号，居中
  - 正文：宋体，四号或小四号
  - 落款：宋体，四号，右对齐

### 二、标点符号

1. **标点使用**
   - 使用中文标点符号
   - 标题末尾不加标点
   - 列举项用分号，最后一项用句号

2. **示例**
   ```
   正确：
   诉讼请求：
   1. 判令被告偿还借款本金10万元；
   2. 判令被告支付利息5000元；
   3. 本案诉讼费用由被告承担。
   ```

### 三、层级编号

```
第一层级：一、二、三、
第二层级：（一）（二）（三）
第三层级：1. 2. 3.
第四层级：（1）（2）（3）
第五层级：①②③
```

## 特殊文书写作要求

### 一、遗嘱

**必备内容**
1. 立遗嘱人基本情况
2. 财产清单（详细列明）
3. 遗产分配方案（明确具体）
4. 立遗嘱时间、地点
5. 立遗嘱人签名

**注意事项**
- 立遗嘱人必须具有完全民事行为能力
- 意思表示必须真实
- 遗嘱内容不得违反法律规定
- 自书遗嘱需全文手写

### 二、授权委托书

**必备内容**
```
授权委托书

委托人：姓名，性别，身份证号，住址
受托人：姓名，性别，身份证号，住址

委托事项：
...

委托权限：
...

委托期限：
...

委托人（签名）：
受托人（签名）：
日期：
```

## 常见问题与示例

### 问题1：诉讼请求不明确

**错误示例**
```
诉讼请求：请求法院依法判决。
```

**正确示例**
```
诉讼请求：
1. 判令被告偿还借款本金人民币10万元；
2. 判令被告支付自2024年1月1日起至实际清偿之日止的利息（按年利率3.45%计算）；
3. 判令被告支付违约金人民币5000元；
4. 本案诉讼费用由被告承担。
```

### 问题2：事实叙述不清

**错误示例**
```
被告借钱不还，我很生气。
```

**正确示例**
```
2024年1月1日，被告以资金周转为由向原告借款人民币10万元，双方签订《借款合同》，约定借款期限为6个月，年利率为3.45%，到期一次性还本付息。原告当日通过银行转账方式向被告支付了借款本金10万元。借款到期后，被告未按约定偿还借款本息，经原告多次催要，被告以各种理由推脱，至今未还款。
```

### 问题3：合同条款不完整

**错误示例**
```
甲方借钱给乙方，乙方要按时还。
```

**正确示例**
```
第一条 借款金额
借款金额为人民币壹拾万元整（￥100,000元）。

第二条 借款期限
借款期限自2025年1月1日起至2025年6月30日止，共6个月。

第三条 利率及利息
借款年利率为3.45%，利息共计人民币1725元，到期一次性支付。

第四条 还款方式
借款到期日，乙方应一次性偿还借款本金及利息共计人民币101,725元。

第五条 违约责任
乙方未按期还款的，应按逾期金额每日万分之五支付违约金。
```

### 问题4：法律引用错误

**错误示例**
```
根据《民法通则》的规定...（注：《民法通则》已废止）
```

**正确示例**
```
根据《中华人民共和国民法典》第六百六十七条规定，借款合同是借款人向贷款人借款，到期返还借款并支付利息的合同。
```

## 总结

法律文书写作的核心要求：
1. **格式规范**：符合官方样式要求
2. **要素完整**：必备要素不遗漏
3. **表述准确**：用词精准，逻辑清晰
4. **引用正确**：法律条文准确有效
