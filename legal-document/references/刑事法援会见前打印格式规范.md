# 刑事法援会见前打印文档格式规范

## 文档概述
刑事法援会见前打印文档是法律援助机构在律师会见嫌疑人/被告人前使用的标准化文书，包含以下五部分：
1. 委托代理/辩护协议
2. 授权委托书
3. 犯罪嫌疑人意思表示
4. 风险告知书
5. 会见笔录

## 模板位置
`../legal_templates/8 刑事法援会见前打印.docx`

## 快速生成命令
```bash
python ../scripts/create_criminal_legal_aid_meeting.py
```

### 5. 空白模板生成

```bash
python ../scripts/create_blank_criminal_legal_aid_meeting.py
```

生成空白模板时，占位符替换为8个空格（`        `），以保证打印文档宽度足够。

### 6. 生成脚本逻辑

```python
from docx import Document

def create_criminal_legal_aid_meeting(weitr, wtrsfz, wtrzz, anyou, jied, quanx="一般代理"):
    """
    生成刑事法援会见前打印文档

    Args:
        weitr: 受援人姓名
        wtrsfz: 受援人身份证号
        wtrzz: 受援人住所/羁押地
        anyou: 涉嫌罪名
        jied: 案件阶段（侦查/审查起诉/审判）
        quanx: 代理权限
    """
    # 模板替换逻辑
    placeholders = {
        'weitr': weitr,
        'wtrsfz': wtrsfz,
        'wtrzz': wtrzz,
        'anyou': anyou,
        'jied': jied,
        'wtxm': '一',  # 默认刑事辩护
        'quanx': quanx
    }

    for para in doc.paragraphs:
        full_text = para.text
        for placeholder, value in placeholders.items():
            if placeholder in full_text:
                for run in para.runs:
                    if placeholder in run.text:
                        run.text = run.text.replace(placeholder, value)
```

## 注意事项
1. 标题使用黑体或方正小标宋简体
2. 正文使用仿宋_GB2312四号
3. 委托书部分使用仿宋_GB2312三号
4. 占位符均为完整占位符，无需分散处理
5. 案件阶段选项：侦查、审查起诉、审判
6. wtxm默认为"一"（刑事辩护）
7. 生成空白模板时，占位符替换为8个空格以保证打印宽度
