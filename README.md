# Trae Skills

Trae IDE 技能集合 - 法律、科技、写作等领域AI助手技能包。

## 技能列表

| 技能名 | 功能 |
|--------|------|
| wewrite-skill | 微信公众号全流程助手 |
| article-writer | 文章写作模块 |
| markdown-converter | Markdown转换 |
| zh-humanizer | 去除AI写作痕迹(中文) |
| hot-topics | 热点抓取 |
| ssh-connect | SSH连接 |
| legal-aid-doc | 法律援助文档 |
| legal-ppt-template | 法律PPT模板 |
| agent-memory | Agent记忆 |
| minimax-docx | Word文档处理 |
| minimax-xlsx | Excel表格处理 |
| minimax-pdf | PDF文档处理 |
| contract-review | 合同审查 |
| legal-document | 法律文书生成 |
| legal-litigation-viz | 诉讼可视化 |
| pptx-generator | PPT生成 |

## 安装

将技能目录复制到 `~/.trae/skills/` 即可使用。
