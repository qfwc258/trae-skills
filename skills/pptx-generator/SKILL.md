# PPTX Generator

## 功能
- PptxGenJS驱动
- 麦肯锡风格支持
- 模块化幻灯片架构
- 专业法律/商业主题

## 设计规范
- 幻灯片尺寸：10" x 5.625" (16:9)
- 字体：Microsoft YaHei / Arial
- 配色：深蓝主色 (#1a5276)

## 触发场景
PPT生成、幻灯片制作、案件汇报、演示文稿
